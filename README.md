# PyQt6_URL_Downloader
Desktop App for URL Downloading based on PyQt6

## Installation:

- **`python -m venv venv`**
- **`venv\Scripts\activate`** or **`source venv/bin/activate`**
- **`pip install -r requirements.txt`**
- **`python app.py`**
